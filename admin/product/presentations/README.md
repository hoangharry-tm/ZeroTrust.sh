# presentations/

This folder contains presentation materials for stakeholder and tech-lead review sessions. Slide decks and exported PDFs go in `exports/`. Visual assets used in presentations (screenshots, architecture diagrams, benchmark charts) go in `assets/screenshots/` and `assets/diagrams/` respectively. Source files for diagrams (e.g., Excalidraw JSON, draw.io XML) should also be stored here so diagrams can be updated without recreating them from scratch.
