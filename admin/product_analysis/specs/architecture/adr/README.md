# specs/architecture/adr/

This folder will hold Architecture Decision Records (ADRs) for ZeroTrust.sh. ADRs are short documents that record a significant technical decision: the context that led to it, the options considered, the chosen option, and the rationale. This folder is intentionally empty during the analysis phase — ADRs are written only after decisions are made. The first ADR will record the architectural approach selection (Approach 1, 2, or 3). Use the standard ADR format: `ADR-NNN-<slug>.md`.
