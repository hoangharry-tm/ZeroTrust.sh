# research/competitor_teardown/

This folder contains competitive intelligence for ZeroTrust.sh. Individual teardown files go in `teardowns/` (one file per competitor, named `<tool_name>_teardown.md`). The aggregated `comparison_matrix.md` at this folder level is the primary deliverable: a side-by-side feature matrix comparing ZeroTrust.sh against Semgrep, Snyk, CodeRabbit, Trufflehog, and Bearer across detection capability, privacy model, performance, cost, and AI-specific threat coverage.
