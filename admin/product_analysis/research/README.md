# research/

This folder contains all externally-sourced research supporting the ZeroTrust.sh product analysis. It is organized into three subfolders: `user_interviews/` for primary developer research, `competitor_teardown/` for competitive intelligence, and `market_analysis/` for market sizing and go-to-market strategy. All claims made in `specs/proposals/` should be traceable to documents in this folder or to cited evidence in `data/evidence/`.
