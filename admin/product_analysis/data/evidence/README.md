# data/evidence/

This folder holds raw supporting evidence for claims made in research documents and proposals: web page exports, academic paper PDFs, screenshots, CSV exports, and link lists. Name files descriptively with a date prefix (e.g., `2026-06-08_pypi_slopsquatting_examples.md`). When a document in `research/` or `specs/` cites a statistic or claim, the source should be traceable to a file here or to an inline URL with an archived copy.
