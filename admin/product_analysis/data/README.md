# data/

This folder contains quantitative evidence and raw data supporting the ZeroTrust.sh analysis. It is divided into `benchmark_results/` for performance and accuracy measurements, and `evidence/` for cited sources, screenshots, and exported datasets. Every factual claim in `specs/proposals/` or `research/` that relies on data should have a corresponding source file here. Benchmark files should follow the naming convention `<date>_<benchmark_name>_results.<ext>`.
