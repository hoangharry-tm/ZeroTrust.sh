# research/user_interviews/

This folder stores all primary research conducted with developers who use AI coding agents. Raw interview notes and transcripts go directly in this folder or in `summaries/` once synthesized. Use `interview_template.md` as the standard question guide for all sessions. The goal is to validate or invalidate the ICP assumptions in `decisions/assumptions.md`, specifically around privacy concerns, pain points with AI-generated code security, and tool adoption willingness.
