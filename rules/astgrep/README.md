# ast-grep Rules

Tree-sitter-based structural rules for languages with weak OpenGrep community packs.
Naming: `<lang>-NNN-<description>.yaml`.

Target languages: Dart, Swift, Rust (Approach 1); Kotlin, C# (Approach 3 via Fraunhofer-AISEC/cpg).

OpenGrep and ast-grep are language-partitioned — they never run the same rules on the same files.
